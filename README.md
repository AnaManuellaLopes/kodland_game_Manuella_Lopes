# 🎮 Space Bot Adventure
### por Ana Manuella dos Santos Lopes

Jogo desenvolvido em Python utilizando **PgZero (Pygame Zero)**.

Projeto criado como atividade prática.  
Faz parte do **Teste para Tutores da Kodland**.

---

## 📋 Requisitos

- Python 3.8 ou superior  
- PgZero  
- Pygame  

---

## ▶️ Comando Principal para Executar

Execute o jogo com um dos comandos abaixo:

### Windows (recomendado)

```bash
py -3.11 -m pgzero game.py

ou

py -m pgzero game.py
```

Linux / macOS
```bash
pgzrun game.py

ou

python3 -m pgzero game.py
````

---


## ⚙️ Instalação

Verifique se o Python está instalado:
python --version
py --version   # Windows

Instale o PgZero:
pip install pgzero

Se não funcionar, tente:
python -m pip install pgzero

ou (Windows):
py -m pip install pgzero
